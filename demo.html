This page is a simple demonstration of The Environmental Data Initiative's Cite service for retrieving citation information from archived data packages.

Enter one or more package IDs from the EDI data portal in the box below. Click the Retrieve Citations button to fetch citations for those packages.

Check your JavaScript console to see the Cite URIs we called and the object we assembled from what came back. Note that our requests include a header that asks for JSON, whereas the URIs in the console will just return the default HTML response.

We only utilize a portion of what Cite can do. See the Cite project for Cite's full capabilities.
